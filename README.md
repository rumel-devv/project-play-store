# 📱 App Store Project (React)

## 🚀 Project Overview

This is a **React-based App Store UI project** where users can:

* Browse different apps
* View detailed information about each app
* See ratings, downloads, and descriptions
* Install apps and view them in an Installed Apps section

The project is designed to mimic a **Google Play Store-style interface** with a clean and responsive UI.

---

## ✨ Features

### 🔍 App Listing

* Display all apps with image, title, rating, and downloads
* Clean card-based UI

### 📄 App Details Page

* Full app information:

  * Image
  * Title & Company Name
  * Description
  * Downloads & Size
  * Ratings breakdown (bar chart style)

### ⭐ Ratings System

* Dynamic rating bars
* Visual representation of user feedback

### 📥 Install App Feature

* Users can install apps
* Installed apps stored using **Context API**

### 📂 Installed Apps Page

* View all installed apps
* Simple and clean list UI
